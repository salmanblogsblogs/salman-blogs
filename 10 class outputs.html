C Programming If Statements Examples, Outputs & Explanations

Posted in 10 Computer Science

Full Examples & Explanations

  1. Example 1 – Simple if 
    #include <stdio.h>
int main() {
    int number = 10;
    if (number > 5) {
        printf("Number is greater than 5");
    }
    return 0;
}

    Output: Number is greater than 5

    Explanation: Since number is 10, the condition number > 5 is true, so the message is printed.

  2. Example 2 – if-else 
    #include <stdio.h>
int main() {
    int marks = 45;
    if (marks >= 50) {
        printf("Pass");
    } else {
        printf("Fail");
    }
    return 0;
}

    Output: Fail

    Explanation: marks is 45, so the condition marks >= 50 is false, and the else block executes.

  3. Example 3 – if-else-if 
    #include <stdio.h>
int main() {
    int marks = 85;
    if (marks >= 90) {
        printf("Grade A+");
    } else if (marks >= 80) {
        printf("Grade A");
    } else if (marks >= 70) {
        printf("Grade B");
    } else {
        printf("Grade C");
    }
    return 0;
}

    Output: Grade A

    Explanation: marks >= 90 → false, marks >= 80 → true, so "Grade A" is printed.

  4. Example 4 – Nested if inside if 
    #include <stdio.h>
int main() {
    int age = 20;
    int hasID = 1;
    if (age >= 18) {
        if (hasID == 1) {
            printf("You can vote");
        } else {
            printf("ID required to vote");
        }
    } else {
        printf("You are underage");
    }
    return 0;
}

    Output: You can vote

    Explanation: First check passes, then inside it hasID == 1 is true, so "You can vote" is printed.

  5. Example 5 – Using if with Logical AND (&&) 
    #include <stdio.h>
int main() {
    int age = 25;
    int citizen = 1;
    if (age >= 18 && citizen == 1) {
        printf("Eligible to vote");
    }
    return 0;
}

    Output: Eligible to vote

    Explanation: Both conditions are true, so the message is printed.

  6. Example 6 – Using if with Logical OR (||) 
    #include <stdio.h>
int main() {
    int marks = 40;
    if (marks < 35 || marks > 100) {
        printf("Invalid marks");
    } else {
        printf("Marks are valid");
    }
    return 0;
}

    Output: Marks are valid

    Explanation: Both OR conditions are false, so else block executes.

  7. Example 7 – if-else-if with Multiple Ranges 
    #include <stdio.h>
int main() {
    int temp = 15;
    if (temp >= 30) {
        printf("Hot day");
    } else if (temp >= 20) {
        printf("Warm day");
    } else if (temp >= 10) {
        printf("Cool day");
    } else {
        printf("Cold day");
    }
    return 0;
}

    Output: Cool day

    Explanation: temp >= 10 is the first true condition, so "Cool day" is printed.

  8. Example 8 – Using if with Equality (==) 
    #include <stdio.h>
int main() {
    int number = 100;
    if (number == 100) {
        printf("Number is exactly 100");
    } else {
        printf("Number is not 100");
    }
    return 0;
}

    Output: Number is exactly 100

    Explanation: Equality operator checks exact match, which is true here.

  9. Example 9 – Odd or Even Check 
    #include <stdio.h>
int main() {
    int num = 7;
    if (num % 2 == 0) {
        printf("Even number");
    } else {
        printf("Odd number");
    }
    return 0;
}

    Output: Odd number

    Explanation: 7 % 2 gives remainder 1, so it's odd.

  10. Example 10 – Nested if with Multiple Checks 
    #include <stdio.h>
int main() {
    int score = 85;
    if (score >= 50) {
        if (score >= 80) {
            printf("Excellent");
        } else {
            printf("Good");
        }
    } else {
        printf("Needs improvement");
    }
    return 0;
}

    Output: Excellent

    Explanation: Both conditions pass, so "Excellent" is printed.

Conclusion

These 10 examples cover the most common uses of if statements in C — from simple conditions to nested checks and logical operators. Mastering these will help you write decision-making code effectively.
