HTML Basics Paragraphs, Links, Images & Lists with MCQs

Posted in 9 Computer Science

Paragraphs

Paragraphs are used in HTML to organize text into readable blocks. Each paragraph starts with <p> and ends with </p>.

Example Code:

<p>This is the first paragraph.</p>
<p>This is the second paragraph.</p>

Output in Browser:

This is the first paragraph.

This is the second paragraph.

Links

Links allow us to jump to another webpage or another part of the same page. Links are created using the <a> tag. The href attribute tells the browser where the link goes. Links can also open your email program when using mailto:.

Example Code:

<a href="https://www.example.com">Visit Example.com</a>
<a href="mailto:info@example.com">Send Email</a>

Images

Images make webpages more attractive and engaging. The <img> tag is used to insert images.

  • src attribute = file path (where the image is).
  • alt attribute = text description (for screen readers or if the image doesn’t load).

Example Code:

<img src="image.jpg" alt="A beautiful scenery">

Lists

Ordered List (Numbered List)

Created using <ol> tag. Each list item goes inside <li>.

<ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>

Output in Browser:

  1. First Item
  2. Second Item
  3. Third Item

Unordered List (Bulleted List)

Created using <ul> tag. Each item goes inside <li>.

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Mango</li>
</ul>

Output in Browser:

  • Apple
  • Banana
  • Mango

Quick Notes – HTML Basics

  • Paragraphs (<p>) – Used to display text in separate blocks. Example: <p>This is a paragraph.</p>
  • Links (<a>) – Used to connect web pages or resources. Syntax: <a href="URL">Text</a>
  • Images (<img>) – Used to insert images on a webpage. Syntax: <img src="image.jpg" alt="description"> (self-closing, no end tag).
  • Ordered List (<ol>) – Displays items in numbered order (1, 2, 3...). Useful for steps or rankings.
  • Unordered List (<ul>) – Displays items with bullets (•). Useful for lists without sequence.

Conclusion

Paragraphs, links, images, and lists are the building blocks of HTML content. Understanding them is essential for creating structured, user-friendly, and visually clear web pages.

MCQs – HTML Basics

  1. Which tag is used to create a paragraph in HTML?
    a) <para>
    b) <pg>
    c) <p>
    d) <text>

    Answer: c) <p>

  2. What does the browser automatically add before and after a paragraph?
    a) A line break only
    b) Space
    c) Bullet points
    d) Nothing

    Answer: b) Space

  3. Which tag is used to create a hyperlink in HTML?
    a) <link>
    b) <a>
    c) <href>
    d) <hyper>

    Answer: b) <a>

  4. What does the href attribute in the <a> tag define?
    a) Font size of link
    b) Background color of link
    c) Destination address of the link
    d) The title of the link

    Answer: c) Destination address of the link

  5. What is the correct syntax to create an email link in HTML?
    a) <a src="email@example.com">Email</a>
    b) <a href="mailto:email@example.com">Email</a>
    c) <a mail="email@example.com">Email</a>
    d) <a email="email@example.com">Email</a>

    Answer: b) <a href="mailto:email@example.com">Email</a>

  6. Which HTML tag is used to display an image?
    a) <picture>
    b) <src>
    c) <img>
    d) <image>

    Answer: c) <img>

  7. What is the purpose of the alt attribute in the <img> tag?
    a) To define the image width
    b) To add an alternative text description
    c) To set the image background
    d) To align the image on the page

    Answer: b) To add an alternative text description

  8. Which of the following is TRUE about the <img> tag?
    a) It is a paired tag.
    b) It does not require a closing tag.
    c) It can only be used inside lists.
    d) It is used for creating links.

    Answer: b) It does not require a closing tag

  9. Which tag is used to create an ordered (numbered) list?
    a) <ul>
    b) <ol>
    c) <li>
    d) <list>

    Answer: b) <ol>

  10. Which tag is used to create an unordered (bulleted) list?
    a) <ul>
    b) <ol>
    c) <li>
    d) <bullet>

    Answer: a) <ul>

  11. What does the <li> tag represent in lists?
    a) List image
    b) List item
    c) Line indent
    d) Link item

    Answer: b) List item
