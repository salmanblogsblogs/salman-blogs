HTML Basic Structure: Definitions, Quick Notes & MCQs

Posted in 9 Computer Science

Basic Structure of an HTML Document

Every HTML document follows a common structure:

  • <!DOCTYPE html> – Declares that the file is HTML5.
  • <html> – Root element; all content must be inside it.
  • <head> – Contains page information (metadata, title, CSS, scripts).
  • <title> – Defines the webpage title (shown in browser tab).
  • <body> – Contains the actual content that is displayed in the browser.
  • <h1> – Defines the largest heading.
  • <p> – Defines a paragraph of text.

HTML Tags

HTML documents are made up of tags. Tags define the structure and content of a webpage.

1. Paired Tags (Container / Open-Close Tags)

They always come in pairs: an opening tag and a closing tag. Closing tag uses a slash /.

  • <p> ... </p> → paragraph
  • <h1> ... </h1> → heading
  • <title> ... </title> → title
  • <div> ... </div> → division block
  • <span> ... </span> → inline container
  • <ul> ... </ul> → unordered list

2. Unpaired Tags (Self-closing / Empty Tags)

They do not require a closing tag. Often used for line breaks, images, input fields, etc.

  • <br> → line break
  • <hr> → horizontal line
  • <img> → insert image
  • <input> → input field
  • <meta> → metadata
  • <link> → link to CSS

Quick Notes – HTML Basic Structure

HTML Tags

HTML documents are made up of tags. Tags define the structure and content of a webpage.

  • Basic HTML structure includes:
    • <!DOCTYPE html> → Declares document type (HTML5).
    • <html> → Root of the document.
    • <head> → Contains metadata, links, scripts, title.
    • <title> → Title shown on browser tab.
    • <body> → Contains visible content.
    • <h1> to <h6> → Headings (h1 is largest, h6 is smallest).
    • <p> → Paragraph of text.
  • HTML Tags Types:
    • Paired Tags = Container Tags / Open-Close Tags (e.g., <p> ... </p>, <div> ... </div>).
    • Unpaired Tags = Self-closing Tags / Empty Tags (e.g., <br>, <hr>, <img>, <input>).
  • Best Practice: Always use proper nesting & indentation for readability.

MCQs – HTML Basic Structure

  1. Which tag is used to declare the type of HTML version in use?
    a) <html>
    b) <head>
    c) <!DOCTYPE html>
    d) <title>

    Answer: c) <!DOCTYPE html>

  2. Which of the following is NOT a paired tag?
    a) <h1>
    b) <p>
    c) <br>
    d) <div>

    Answer: c) <br>

  3. Which tag is an example of a self-closing tag?
    a) <p>
    b) <img>
    c) <h1>
    d) <title>

    Answer: b) <img>

  4. The content inside the <title> tag appears where?
    a) Inside the webpage
    b) Browser tab
    c) Bottom of the page
    d) Console

    Answer: b) Browser tab

  5. Which of the following are examples of paired tags?
    a) <ul> ... </ul>
    b) <br>
    c) <hr>
    d) <img>

    Answer: a) <ul> ... </ul>

  6. The <meta> tag belongs to which type?
    a) Paired Tag
    b) Self-closing Tag
    c) Nested Tag
    d) None

    Answer: b) Self-closing Tag
