C Programs with Variables

Posted in 10 Computer Science

Single Variable Programs

1. Program with int

#include <stdio.h>
int main() {
    int age = 22;
    printf("My age is %d\n", age);
    return 0;
}

Output:
My age is 22

2. Program with float

#include <stdio.h>
int main() {
    float height = 5.9;
    printf("My height is %.1f feet\n", height);
    return 0;
}

Output:
My height is 5.9 feet

3. Program with char

#include <stdio.h>
int main() {
    char grade = 'A';
    printf("My grade is %c\n", grade);
    return 0;
}

Output:
My grade is A

Two Variables Programs

4. Program with int and float

#include <stdio.h>
int main() {
    int rollNo = 101;
    float marks = 87.5;
    printf("Roll No: %d\n", rollNo);
    printf("Marks: %.1f\n", marks);
    return 0;
}

Output:
Roll No: 101
Marks: 87.5

5. Program with int and char

#include <stdio.h>
int main() {
    int age = 20;
    char grade = 'B';
    printf("Age: %d\n", age);
    printf("Grade: %c\n", grade);
    return 0;
}

Output:
Age: 20
Grade: B

Three Variables Program

6. Program with int, float, and char

#include <stdio.h>
int main() {
    int id = 123;
    float salary = 55000.75;
    char grade = 'A';
    
    printf("Employee ID: %d\n", id);
    printf("Salary: %.2f\n", salary);
    printf("Grade: %c\n", grade);
    return 0;
}

Output:
Employee ID: 123
Salary: 55000.75
Grade: A

Programs with Same Type Variables

1. Using int x, y, z

#include <stdio.h>
int main() {
    int x = 10, y = 20, z = 30;
    printf("Value of x: %d\n", x);
    printf("Value of y: %d\n", y);
    printf("Value of z: %d\n", z);
    return 0;
}

Output:
Value of x: 10
Value of y: 20
Value of z: 30

2. Using float a, b, c

#include <stdio.h>
int main() {
    float a = 2.5, b = 4.8, c = 6.3;
    printf("Value of a: %.1f\n", a);
    printf("Value of b: %.1f\n", b);
    printf("Value of c: %.1f\n", c);
    return 0;
}

Output:
Value of a: 2.5
Value of b: 4.8
Value of c: 6.3

3. Using char ch1, ch2, ch3

#include <stdio.h>
int main() {
    char ch1 = 'A', ch2 = 'B', ch3 = 'C';
    printf("First character: %c\n", ch1);
    printf("Second character: %c\n", ch2);
    printf("Third character: %c\n", ch3);
    return 0;
}

Output:
First character: A
Second character: B
Third character: C

Programs with Same Type Variables (Single Line printf)

1. Using int x, y, z

#include <stdio.h>
int main() {
    int x = 10, y = 20, z = 30;
    printf("x = %d, y = %d, z = %d\n", x, y, z);
    return 0;
}

Output:
x = 10, y = 20, z = 30

2. Using float a, b, c

#include <stdio.h>
int main() {
    float a = 2.5, b = 4.8, c = 6.3;
    printf("a = %.1f, b = %.1f, c = %.1f\n", a, b, c);
    return 0;
}

Output:
a = 2.5, b = 4.8, c = 6.3

3. Using char ch1, ch2, ch3

#include <stdio.h>
int main() {
    char ch1 = 'A', ch2 = 'B', ch3 = 'C';
    printf("Characters: %c, %c, %c\n", ch1, ch2, ch3);
    return 0;
}

Output:
Characters: A, B, C

Practice Statements

  • Declare 3 integer variables a, b, c with any values and print them in a single line using printf.
  • Declare 2 float variables price and discount, assign values, and print them with 1 decimal place.
  • Declare 3 char variables ch1, ch2, ch3 (e.g., 'X', 'Y', 'Z') and print them in one line.
  • Declare an integer rollNo, a float marks, and a char grade, and print them in one printf.
  • Declare 2 integer variables by any name and print only the second one.

MCQs on Data Types (int, float, char)

  1. Which of the following is the correct format specifier for int in C?
    a) %d
    b) %f
    c) %c
    d) %s

    Answer: a) %d

  2. What is the size of int in most 32-bit compilers?
    a) 1 byte
    b) 2 bytes
    c) 4 bytes
    d) 8 bytes

    Answer: c) 4 bytes

  3. Which of the following can store decimal (fractional) values in C?
    a) int
    b) char
    c) float
    d) double

    Answer: c) float

  4. What is the correct format specifier for float in printf?
    a) %f
    b) %d
    c) %c
    d) %lf

    Answer: a) %f

  5. Which data type is best to store a single character like 'A'?
    a) int
    b) char
    c) float
    d) double

    Answer: b) char

  6. What is the output of the following code?
    #include <stdio.h>
int main() {
    char ch = 'X';
    printf("%c", ch);
    return 0;
}
    a) X
    b) x
    c) 88
    d) Error

    Answer: a) X

  7. What will be the output of this program?
    #include <stdio.h>
int main() {
    int a = 5;
    float b = 3.2;
    printf("%d %.1f", a, b);
    return 0;
}
    a) 5 3.2
    b) a b
    c) 5 3
    d) Error

    Answer: a) 5 3.2

  8. Which of the following is true about char in C?
    a) It can store multiple letters
    b) It always stores ASCII values of characters
    c) It is 4 bytes in size
    d) It is same as float

    Answer: b) It always stores ASCII values of characters

  9. Which of these can represent both positive and negative numbers?
    a) int
    b) char
    c) float
    d) All of the above

    Answer: d) All of the above

  10. Which of the following is invalid in C?
    a) int age = 20;
    b) float price = 99.99;
    c) char name = "Ali";
    d) char grade = 'A';

    Answer: c) char name = "Ali"; (A char can only store one character, not a string).
